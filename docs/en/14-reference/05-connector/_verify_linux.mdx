In the Linux shell, you can directly execute `taos` to connect to the TDengine service and enter the TDengine CLI interface. Here is an example:

```text
$ taos

taos> show databases;
              name              |
=================================
 information_schema             |
 performance_schema             |
 db                             |
Query OK, 3 rows in database (0.019154s)

taos>
```
