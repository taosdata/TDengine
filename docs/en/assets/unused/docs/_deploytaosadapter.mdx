### Deploying taosAdapter

taosAdapter can be deployed on the same system as TDengine or separately.

Start taosAdapter:

```
systemctl start taosadapter
```

Check the running status of taosAdapter:

```
systemctl status taosadapter
```

For detailed configuration parameters and usage of taosAdapter, please refer to the output of the `taosadapter --help` command and the [reference documentation](../../reference/taosadapter).
