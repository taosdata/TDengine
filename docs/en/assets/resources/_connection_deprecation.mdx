:::warning Deprecation Notice

**<font color="red">The following connection methods are deprecated and will be discontinued on 2027-01-01</font>**

##### Native Connection (Java, C#, Go, ODBC)

Please migrate to WebSocket connection:

- **Java**: Change `jdbc:TAOS://` to `jdbc:TAOS-WS://`, port from `6030` to `6041`. For other configurations, please refer to [JDBC Driver](../../14-reference/05-connector/14-java.md#jdbc-driver).
- **C#**: Change `protocol=Native` to `protocol=WebSocket` in the connection string, port from `6030` to `6041`.
- **ODBC**: In the ODBC data source configuration, change the connection type from `Native` to `WebSocket`, and change the server address to URL format (e.g., `http://localhost:6041`). See [ODBC Connector](../../14-reference/05-connector/50-odbc.md) for details.
- **Go**:

    <details>
    <summary>Go Native to WebSocket Migration Guide</summary>

  ##### 1. Execute SQL

    `database/sql` code requires no modification, mainly replace driver name and DSN protocol header: from `taosSql` + `tcp(6030)` to: `taosWS` + `ws(6041)`.

  ##### 2. Data Subscription

    Migrate from Native's `af/tmq` to `ws/tmq`.

  ##### 3. STMT and Schemaless

    Migrate from `af` to `ws/unified`.

    </details>

C/C++, Python, and Rust native connections will continue to be supported, but **still recommend using WebSocket connection** for better compatibility.

##### REST Connection (Java, Python, Go)

Please migrate to WebSocket connection:

- **Java**: Change `jdbc:TAOS-RS://` to `jdbc:TAOS-WS://`. For other configurations, please refer to [JDBC Driver](../../14-reference/05-connector/14-java.md#jdbc-driver).
- **Go**: `database/sql` code requires no modification, mainly replace driver name and DSN protocol header: from `taosRestful` + `http(6041)` to `taosWS` + `ws(6041)`.

- **Python**:
    <details>
    <summary>Python taosrest to taos-ws-py Migration Guide</summary>

  ##### Step 1: Replace Connection Method

    `taosrest` typically uses `url + user/password` parameters to connect:

    ```python
    from taosrest import connect

    conn = connect(url="http://localhost:6041", user="root", password="taosdata")
    ```

    Migrate to `taosws` WebSocket DSN:

    ```python
    import taosws

    conn = taosws.connect("ws://root:taosdata@localhost:6041")
    ```

  ##### Step 2: Replace SQL Execution Method

    Common `taosrest` usage is through cursor:

    ```python
    cursor = conn.cursor()
    cursor.execute("CREATE DATABASE IF NOT EXISTS demo")
    cursor.execute("USE demo")
    cursor.execute("CREATE TABLE IF NOT EXISTS t (ts TIMESTAMP, v INT)")
    cursor.execute("INSERT INTO t VALUES (now, 1)")
    ```

    When migrating to `taosws`, you can continue using cursor style for minimal changes:

    ```python
    cursor = conn.cursor()
    cursor.execute("CREATE DATABASE IF NOT EXISTS demo")
    cursor.execute("USE demo")
    cursor.execute("CREATE TABLE IF NOT EXISTS t (ts TIMESTAMP, v INT)")
    cursor.execute("INSERT INTO t VALUES (now, 1)")
    ```

  ##### Step 3: Replace Query Result Reading Method

    Common `taosrest` usage:

    ```python
    cursor.execute("SELECT * FROM demo.t")
    rows = cursor.fetchall()
    for row in rows:
        print(row)
    ```

    When reading query results with `taosws`, you can also keep cursor style:

    ```python
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM demo.t")
    rows = cursor.fetchall()
    for row in rows:
        print(row)
    ```

    </details>
