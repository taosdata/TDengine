## 特性
  1. 特性：支持 OPC 冗余通信

## 优化
  1. 优化：提升虚拟超级表的查询性能
  2. 优化：提升 last_row + composite key 的查询性能
  3. 优化：提升 last_row + tags 的查询性能
  4. 优化：不再重试已经超时的 RPC 消息
  5. 优化：降低流计算在读取数据时的 CPU 使用比例
  6. 优化：改进 mnode 启动时的日志记录和错误码显示
  7. 优化：增加配置参数 rpcRecvLogThreshold，用于设置 RPC 模块警告日志的阈值
  8. 优化：优化错误码列表文档
  9. 优化：taosX 允许从高精度库导入到低精度库
 10. 优化：Adapter 支持连接管理和请求限制
 11. 优化：taosws taos_stmt_num_params 支持查询
 12. 优化：C/Rust WebSocket 连接器小查询性能优化
 13. 优化：REST API 返回 JSON nan/inf 为 NULL
 14. 优化：Explorer Grid 绘图支持 DECIMAL 数据类型
 15. 优化：Explorer dashboard 表支持排序
 16. 优化：OpenTSDB 导入支持自定义列名
 17. 优化：统一企业版和社区版的登录注册逻辑
 
## 修复
  1. 修复：订阅元数据消息的标签包含空字符串时，tmq_get_json_meta 表现不符合预期
  2. 修复：WAL 版本号超过 int32 最大值时导致的数据订阅失败问题
  3. 修复：数据同步消息发送失败时，因清理内存操作导致的崩溃问题
  4. 修复：解析 overflow 页可能返回错误的页号
  5. 修复：并发写入与 last 查询同时进行时，出现极小概率崩溃的问题
  6. 修复：更新服务端配置参数后无法立即查询到新参数值的问题
  7. 修复：当存在 RSMA 时，删除并创建同名数据库时 retention 任务不退出
  8. 修复：taos_options_connection 清空 user_ip 不生效的问题
  9. 修复：双副本单节点断网后选主过慢的问题
 10. 修复：修改 dnode 配置参数后各 dnode 配置参数值小概率不一致的问题
 11. 修复：流计算语句包含 tag=%%1时计算结果错误
 12. 修复：状态窗口未能正确处理 NULL 值数据
 13. 修复：虚拟超级表查询中因未包含时间戳主键而导致的系统崩溃问题
 14. 修复：在 200 核及以上的 CPU 上服务器启动失败的问题
 15. 修复：窗口查询中的无效 sttblockreader 初始化动作
 16. 修复：perf_connections 中 end_point 信息不正确的问题
 17. 修复：带有滑动（sliding）的时间窗口查询中，_wduration 结果错误的问题
 18. 修复：当 tag 值为空字符串时，show create table 命令显示错误
 19. 修复：taos-connector-rust 的 free(): invalid pointer
 20. 修复：修复 C WebSocket stmt2 自动重连
 21. 修复：Explorer 查询 NAN 数据报错
