## 优化

1. taos-explorer 创建 taosX 任务时，若无可用 XNODE，将自动引导至 XNODE 创建页面
2. taos-explorer DataIn 创建超级表时，若将数据列移至标签列，将自动删除默认空标签列
3. 增强 taos-explorer 跨域安全
4. 新增 taos-explorer 的 XNODE 管理页面
5. 优化 taosX 与 xnoded 之间的通信日志
6. 优化 taos-ws-py 连接器的 SQLAlchemy 适配
7. taosX 支持通过 taos-explorer 导入和导出配置文件
8. taosX 的 AVEVA Historian 数据源支持断开后自动重连
9. Rust 连接器支持订阅批量标签修改消息
10. Rust 连接器的参数绑定支持 DECIMAL 和 BLOB 类型
11. 新增连接器安全开发指南与安全示例
12. taos shell 新增 -H 选项，支持将包含不可打印字符的 BINARY 字符串以十六进制格式（0x...）显示，默认关闭
13. to_iso8601 函数支持夏令时转换
14. to_char 和 to_timestamp 函数的 TZH 格式支持指定分钟级时区偏移
15. STMT2 写入模式下，子表元数据变更时不再返回 TSDB_CODE_NEED_RETRY 错误
16. interval 窗口支持使用客户端携带的时区进行划分
17. 优化 show table distribute 的返回格式，并新增 INFORMATION_SCHEMA.INS_DISK_USAGE 系统表，用于查询表数据分布情况
18. 优化 WAL 最后一个日志文件损坏时的截断处理逻辑
19. 允许删除被订阅使用的表；删除并重建表后，执行 reload topic 即可恢复订阅
20. 新增配置参数 syncAssignedCheckAppliedGap，在双副本模式下，当节点被指定为 assigned leader 时，用于控制 follower 的 applied index 与 commit index 之间的差距阈值，避免因 follower 同步进度未追上而进入 restoring 状态
21. 虚拟表支持 DECIMAL 类型列

## 修复

1. 修复 taos-explorer 数据写入任务较多时页面加载缓慢的问题
2. 修复 taos-explorer 返回 SQL 执行时间不准确的问题
3. 修复 taosX 在 taos-explorer 中进行任务配置时过滤条件不生效的问题
4. 修复 taosX 在无数据场景下生成备份点异常的问题
5. 修复 taosX 在绑定 6051 端口时，因内部逻辑错误导致 Windows 平台 OPC 接入失败的问题
6. 修复 taosX 的 Pulsar 数据源选择自定义认证时未显示认证参数输入框的问题
7. 修复 taosX 的 Pulsar 数据源因不支持数据切分而持续处于创建中状态的问题
8. 修复 taosX 的 Kafka 数据源在生成带复合主键的时间戳列数据时错误使用自增方式的问题
9. 修复 taosX 的 SparkPlugB 获取示例数据时，发生错误后无提示且无法重新拉取的问题
10. 修复 xnoded 子进程在 Windows 平台概率性无法停止的问题
11. 修复 taosdump 通过 WebSocket 读取数据时，在引擎服务异常后返回空数据但未抛出异常的问题
12. 修复 taosdump 导出大数据量子表时性能显著下降的问题
13. 修复 taosk 工具生成密钥时路径错误的问题
14. 修复通过 taosAdapter 访问时 queryTableNotExistAsEmpty 参数不生效的问题
15. 修复纳秒精度数据库中，流计算输出表的 TAG 列使用固定值时建流语句报错的问题
16. 修复数据库过期数据删除（trim）与共享存储迁移操作冲突的问题
17. 修复数据库重整时，若部分节点长时间离线，可能导致 mnode 消息队列打满并引发连接异常的问题
18. 修复监控信息写入日志库失败的问题
19. 修复虚拟子表与虚拟超级表 Schema 中列数不一致的隐患
20. 修复查询语句中部分函数在无 FROM 子句时执行导致崩溃的问题
21. 修复查询语句中包含 not (cast(null as varbinary) between now() and ts) 这类类型转换表达式时导致程序崩溃的问题
22. 修复流计算结果表为已存在普通表时，写入计算结果报错的问题
23. 修复流计算查询语句中使用 %%tbname 时，在大数据写入场景下因内存释放逻辑错误引发 OOM 的问题
24. 修复流计算计数窗口触发在读取历史数据时内存占用过高的问题
25. 修复流计算回放 WAL 时，因未限制滑动窗口数量上限导致的 OOM 崩溃问题
26. 修复流计算外层查询中 diff() 与 notify 组合使用导致的崩溃问题
27. 修复 group_concat 函数崩溃及结果错误的问题
28. 修复 CASE WHEN 语句引用主键作为过滤条件时，执行计划与实际执行逻辑不一致的问题
29. 修复 taos_cleanup 退出时的内存泄漏问题
30. 修复 macOS 平台 USE DATABASE 语句输出提示与 Linux 不一致的问题，统一显示为 Database changed.
31. 修复 Windows 安装包无法优雅关闭进程的问题
