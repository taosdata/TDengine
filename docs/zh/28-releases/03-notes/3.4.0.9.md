## 特性

  1. 特性：taosAdapter 订阅支持 TOKEN
  2. 特性：taosdump 支持虚拟表导出导入

## 优化

  1. 优化：Explorer UI 大量优化
  2. 优化：海量子表存在订阅时，频繁修改子表标签对数据写入的影响
  3. 优化：超级表模板名和子表名支持表达式计算
  4. 优化：提升虚拟表查询含 tbname 列时的性能
  5. 优化：减少默认日志保存个数为 3
  6. 优化：将配置参数 supportVnodes 的最大值从 4096 调整为 1024
  7. 优化：CREATE XNODE TASK 报错信息优化

## 修复

  1. 修复：Kafka 连通性检查添加日志打印
  2. 修复：对特定流授予 GRANT ALL 权限后不生效的问题
  3. 修复：赋权 START/STOP STREAM 权限后，权限不生效的问题
  4. 修复：拥有 DROP STREAM 权限时，仍无法删除流的问题
  5. 修复：查询无权限的视图导致程序崩溃的问题
  6. 修复：授权 ALTER VIEW 权限后，权限不生效的问题
  7. 修复：审计员查看用户完整信息失败的问题
  8. 修复：流计算中读取含 varchar 类型复合主键的表时，获取时间戳触发服务崩溃的问题
  9. 修复：使用 IF EXISTS 语法删除角色时报错的问题
 10. 修复：权限缺失时报错信息不明确的问题
 11. 修复：macOS 系统下调用原生连接的 taos_query 方法时，端口占用持续增加的问题
 12. 修复：流计算子查询中 JOIN 语句以_twstart 等时间占位符为查询结果第一列时，误报无时间戳主键的错误
 13. 修复：流计算子查询含 interval 窗口时，查询结果的时间戳列设置错误
 14. 修复：未设置加密密钥时，审计数据库创建操作无返回结果的问题
 15. 修复：普通用户创建普通表后，未显式授权则无权限写入的问题
 16. 修复：将新增加密函数名改为关键字，解决数据写入时关键字误作内容写入数据库的问题
 17. 修复：timetruncate 函数偶发不返回结果的问题
 18. 修复：create xnode 时意外的错误信息
 19. 修复：taosX 任务 Parser 长度扩展为 48k

 23. 修复：Explorer 切换共享主题时报语法错误

 26. 修复：MySQL 不支持 tinyint(1)/boolean 类型
